### 3.1 InstaTripCard (인스타그램 스타일 카드)
- **기능**: 선정된 여행지의 상세 정보를 시각적으로 표현.

  - 본문: 사용자 캡션 및 해시태그.

- **기능**: 버튼 클릭 시 랜덤하게 여행지를 선정하여 표시 (현재 Ver.2 구현을 위해 주석 처리됨).
- **로직**: `setTimeout`을 이용한 로딩 애니메이션 효과 구현.

## 4. 트러블슈팅 및 작업 내역 (2026.04.16)

### 4.1 ESLint 설정 고도화
- **문제**: VS Code에서 `no-unused-vars` 에러(빨간 줄)가 사라지지 않음. JSX를 사용하는 컴포넌트를 ESLint가 제대로 인식하지 못함.
- **해결**: 
  - `eslint-plugin-react` 패키지 설치.
  - `eslint.config.js` 파일에 `react` 플러그인 추가 및 `jsx-runtime` 설정 적용.
  - React 버전을 자동 감지하도록 `settings` 추가.

### 4.2 App.jsx 구조 개선
- **문제**: `App` 함수 내부에 `InstaTripCard`가 중첩 선언되어 있었으며, `App` 함수가 아무것도 반환하지 않아 렌더링 오류 발생.
- **해결**: `App` 함수가 `InstaTripCard`를 반환하도록 구조를 변경하고, 불필요한 중괄호(`}`)를 정리하여 문법 오류 해결.

### 4.3 JSX 문법 오류 수정
- **문제**: JSX 내부에서 일반 JS 주석(`/* ... */`)을 사용하여 텍스트 노드로 인식되는 `react/jsx-no-comment-textnodes` 에러 발생.
- **해결**: 모든 JSX 내부 주석을 `{/* ... */}` 형식으로 수정 완료.
